# Installation

## Arch linux

`git clone https://github.com/PierreLeBlond/Orangutan.git --recursive` \
or \
`git clone https://github.com/PierreLeBlond/Orangutan.git` \
`git submodule update --init --recursive`

```
cmake .
make
./Orangutan
```
